百度搜索 VcXsrv

下载地址：https://sourceforge.net/projects/vcxsrv/

用win的文件管理器打开linux目录的方法

在文件管理器地址中输入 \\wsl$\<子系统名>\ 回车，即可进入子系统的根目录下，比如Ubuntu，就是\\wsl$\Ubuntu\

在linux终端中访问win的文件

  • 方法一：使用linux终端，cd到 /mnt/ 目录下，里面有 c,d,e,…等目录，那些对应win的盘符，直接cd进去即可。
  • 方法二：在win的文件管理器中打开对应的目录，然后按住 shift 键的同时鼠标右击目录空白处，选择 在此处打开Linux shell。

开发工具

MinGW安装

mingw 是一个开发工具链，包含了 gcc 编译器以及工具 make

下载地址：https://sourceforge.net/projects/mingw-w64/files/mingw-w64/

make 命令需要把 mingw 安装目录下的 bin 目录中的 mingw32-make.exe 改名为 make.exe ，复制一份改好名后放回去也行。

CMake安装

cmake 是一个项目构建工具，可用其生成 Makefile，著名的集成开发环境 CLion 就是基于 cmake 来组织项目的。

下载地址：https://cmake.org/download/

可以下载压缩包版本，手动配置环境变量。

openjdk11安装

下载地址：http://jdk.java.net/

注意：这个网站的 openjdk8 win版本只有 i586 的（只有32位），可以考虑 openjdk11，或者下载源代码自己编译。

下载好后解压便是一个完整的jdk，放到喜欢的位置后，配置环境变量即可。

llvm

LLVM项目是模块化、可重用的编译器以及工具链技术的集合，里面有 clang，clang-format，clangd 等工具。

下载地址：

  • 官网：https://releases.llvm.org/
  • github：https://github.com/llvm/llvm-project/releases

建议下载release版本，不要下载rc版本（Release Candidate）

附上一个 clang-format 配置文件（clang-format具体怎么用，自行百度，或者使用vscode内部的设置进行简单的调整）

vscode

下载地址：https://code.visualstudio.com/#alt-downloads

建议下载 System Installer 。

我的插件列表（按需安装，适当关闭，避免vscode太卡）：

  • Chinese (Simplified) Language Pack for Visual Studio Code （中文语言包）
  • Dracula Official （德古拉主题）
  • Markdown All in One （写Markdown必备）
  • Visual Studio IntelliCode （基于AI的代码提示）
  • Vim （vim插件）
  • C/C++ （C/C++语言支持）
  • Clang-Format （C/C++代码格式化）
  • clangd （使用clangd的代码补全）
  • CMake （cmake语言支持）
  • CMake Tools （cmake扩展工具）
  • HTML CSS Support （html和css语言包支持）
  • HTML Snippets （html代码块）
  • JavaScript (ES6) code snippets （js代码块）
  • Live Server （边修改边预览，web前端开发必备）
  • Java Extension Pack （java开发扩展包，里面包含了好多子插件）
  • Python （python语言支持，需要电脑上先安装python，并配置环境变量）
  • Python Extended （python开发扩展包）
  • Remote - WSL （用于vscode连接子系统进行开发）
  • Remote - SSH （用于使用ssh连接远程主机开发）
  • XML Tools （xml语言支持）
  • MASM （masm汇编语法高亮）
  • ASM Code Lens （用于辅助汇编语言开发）

vscode自带的编译和运行是以文件夹为单位的而不是单个文件，因此需要打开文件夹，然后配置C/C++的运行和调试，具体操作自行百度即可。如果需要只运行一段简单的代码，可以考虑 Code Runner 插件，或者用 Ctrl + ` 来打开终端，用命令行来编译运行。
